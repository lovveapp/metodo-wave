# O Método Wave

## Conceito central

O Wave organiza o treino de força como um sistema com variáveis controláveis, onde o padrão de distribuição dos grupos musculares ao longo das sessões forma uma onda — daí o nome. Quando visualizado em um gráfico (sessões × grupos musculares × volume), a distribuição não é linear nem aleatória: é periódica, com cada grupo aparecendo em momentos calculados para maximizar recuperação e volume acumulado.

## Princípios de design

### 1. Intervalo de descanso linear entre grupos musculares
A distribuição dos grupos nas sessões segue uma lógica onde o intervalo entre dois estímulos no mesmo grupo é previsível e suficiente. Não é apenas "não treinar o mesmo músculo dois dias seguidos" — é garantir que o padrão de recuperação seja matematicamente consistente ao longo de semanas.

### 2. Ênfase ajustável
O modelo permite aumentar ou reduzir a frequência e/ou volume de um grupo específico sem quebrar a estrutura dos demais. Isso é importante porque diferentes pessoas têm prioridades diferentes (ex: mais enfase em quadríceps, menos em panturrilha) e essas prioridades mudam ao longo do tempo.

### 3. Sequenciamento intrassessão
Dentro de cada sessão, a ordem dos exercícios é planejada para que o grupo muscular sendo treinado esteja no seu pico de recuperação relativa naquele momento. Isso reduz a interferência entre grupos e melhora a qualidade do estímulo por exercício.

### 4. Gestão de fadiga por sessão
O Wave é desenhado para que o praticante termine cada sessão cansado, mas não destruído. A sessão seguinte deve ser executável com qualidade. Isso é diferente de "deixar algo no tanque por precaução" — é uma consequência do sequenciamento e do volume calculado.

### 5. Frequência mínima de 2x por semana por grupo
Alinhado com a literatura científica, cada grupo muscular é treinado pelo menos 2 vezes por semana. A frequência emerge da estrutura do modelo, não é uma regra imposta sobre ela.

### 6. Volume semanal dentro da faixa ótima
O volume total de séries por grupo por semana é calculado para ficar dentro do intervalo evidenciado como ótimo para hipertrofia: entre 10 e 20 séries semanais por grupo, dependendo da prioridade.

### 7. Ciclos de exercícios por ângulo e padrão de movimento
O Wave não apenas distribui grupos musculares — ele prescreve exercícios pensando em variar o ângulo de trabalho e o padrão de movimento dentro e entre sessões. Por exemplo: numa sessão de peito, o modelo pode combinar supino inclinado (ângulo inclinado, empurrar) + crucifixo na polia alta (ângulo descendente, adução). São ângulos e padrões diferentes para o mesmo grupo.

Essa abordagem tem suporte científico direto: a variação sistemática por critérios anatômicos e biomecânicos promove adaptações hipertróficas regionais superiores em comparação a repetir o mesmo padrão de movimento. A palavra-chave é *sistemática* — variação aleatória ou excessiva pode prejudicar os ganhos. O Wave define os critérios de variação, não deixa a escolha ao acaso.

## Estrutura das sessões

O Wave é organizado em um ciclo de 4 sessões. O nome de cada sessão tem dois componentes:

- **Primeiro nome (Upper / Lower):** região do corpo trabalhada naquela sessão
- **Segundo nome (Front / Back):** ênfase do dia — qual cadeia muscular recebe maior estímulo

```
Ciclo Wave:
  1. Upper Front  (UF)
  2. Lower Front  (LF)
  3. Upper Back   (UB)
  4. Lower Back   (LB)
```

### Upper Front — sequência base

| Ordem | Grupo muscular         |
|-------|------------------------|
| 1     | Peito                  |
| 2     | Costas                 |
| 3     | Peito                  |
| 4     | Ombro anterior         |
| 5     | Tríceps                |
| 6     | Bíceps                 |
| 7     | Tríceps                |
| 8     | Abdômen                |

### Upper Back — sequência base (ênfase invertida)

| Ordem | Grupo muscular         |
|-------|------------------------|
| 1     | Costas                 |
| 2     | Peito                  |
| 3     | Costas                 |
| 4     | Ombro posterior        |
| 5     | Bíceps                 |
| 6     | Tríceps                |
| 7     | Bíceps                 |
| 8     | Abdômen                |

### Lower Front — sequência base

| Ordem | Grupo muscular         |
|-------|------------------------|
| 1     | Quadríceps             |
| 2     | Posterior              |
| 3     | Quadríceps             |
| 4     | Adutor                 |
| 5     | Panturrilha            |
| 6     | Panturrilha            |
| 7     | Ombro lateral          |

### Lower Back — sequência base (ênfase invertida)

| Ordem | Grupo muscular         |
|-------|------------------------|
| 1     | Posterior              |
| 2     | Quadríceps             |
| 3     | Posterior              |
| 4     | Glúteo / Abdutor       |
| 5     | Panturrilha            |
| 6     | Panturrilha            |
| 7     | Ombro lateral          |

### Detalhe: ombro lateral nos dias de Lower

O ombro lateral aparece em todas as sessões Lower. A distribuição ao longo do ciclo fica:

| Feixe          | Sessão               | Frequência/ciclo |
|----------------|----------------------|------------------|
| Anterior       | Upper Front          | 1x               |
| Posterior      | Upper Back           | 1x               |
| Lateral        | Lower Front + Lower Back | 2x           |

Proporção resultante: **1 : 1 : 2** (anterior : posterior : lateral).

Essa proporção não é arbitrária — é uma consequência direta da biomecânica dos treinos de Upper:

- No **Upper Front**, o deltóide anterior já é recrutado como músculo sinergista nos exercícios de peito (supino, crucifixo). Ele já recebe estímulo indireto nessa sessão.
- No **Upper Back**, o deltóide posterior já é recrutado como sinergista nos exercícios de costas (remadas, puxadas). Ele também já recebe estímulo indireto.
- O **deltóide lateral** não é sinergista primário de nenhum dos dois padrões de movimento principais. Ele não recebe estímulo indireto nos dias de Upper — por isso precisa de estímulo direto 2x por ciclo, nos dias de Lower.

O Wave não "dobra" o lateral por preferência estética — dobra porque é o único feixe que não aparece como sinergista em nenhum dia de Upper. A proporção 1:1:2 emerge naturalmente da análise biomecânica dos movimentos.

### O padrão intrassessão

As sessões Upper e Lower têm comprimentos diferentes (8 e 6 exercícios, respectivamente) mas seguem a mesma lógica estrutural:

**Upper (8 exercícios):**
- **Posições 1–3:** primário → antagonista → primário. O antagonista serve de descanso ativo entre as séries do grupo principal.
- **Posição 4:** ombro com ênfase do dia (anterior no Front, posterior no Back).
- **Posições 5–7:** mesma alternância, aplicada aos braços (tríceps/bíceps).
- **Posição 8:** abdômen fixo ao final.

**Lower (6 exercícios):**
- **Posições 1–3:** primário → antagonista → primário (quadríceps/posterior, invertido no Back).
- **Posição 4:** músculo hip-focused — adutor no Front, glúteo/abdutor no Back.
- **Posição 5:** panturrilha fixo.
- **Posição 6:** ombro lateral fixo (ver seção de proporcionalidade acima).

Em ambos os casos, o grupo muscular que vai ser trabalhado chegou àquela posição com o máximo de recuperação possível dentro da sessão — que é o objetivo do sequenciamento intrassessão.

### Front vs. Back: a dualidade que cria a onda

A inversão de ênfase entre UF e UB é o mecanismo central que gera o padrão de "onda" no gráfico. Peito aparece 3x no UF e 1x no UB; costas aparece 1x no UF e 3x no UB. Quando a semana tem os dois treinos, cada grupo recebe volume equivalente — mas a distribuição oscila entre as sessões. É essa oscilação que, visualizada ao longo de várias semanas, forma a curva característica do método.

## Frequência semanal e estrutura de macrociclo

O Wave é desenhado para quem treina **5 ou 6 vezes por semana**. A frequência escolhida muda como o ciclo de 4 sessões se distribui ao longo das semanas — e isso cria padrões de volume diferentes com implicações diretas para recuperação.

### 6 dias por semana

Com 6 sessões semanais, o ciclo de 4 treinos se repete uma vez e meia por semana:

| Semana | Sessões                          | Balanço      |
|--------|----------------------------------|--------------|
| 1      | UF · LF · UB · LB · UF · LF     | 4 Front, 2 Back |
| 2      | UB · LB · UF · LF · UB · LB     | 4 Back, 2 Front |
| 3      | UF · LF · UB · LB · UF · LF     | 4 Front, 2 Back |
| ...    | (alterna a cada semana)          |              |

Resultado: o volume semanal por cadeia muscular alterna entre semanas mais pesadas e semanas mais leves de forma automática. A semana com 4 sessões Back funciona como mini deload para os grupos Front — e vice-versa. Esse efeito não é programado manualmente: é uma consequência da estrutura do ciclo.

### 5 dias por semana

Com 5 sessões semanais, o ciclo avança 1 sessão por semana, criando um macrociclo de 4 semanas onde cada semana tem uma ênfase diferente:

| Semana | Sessões                       | Ênfase da semana     |
|--------|-------------------------------|----------------------|
| 1      | UF · LF · UB · LB · **UF**   | Upper Front (abre e fecha a semana) |
| 2      | LF · UB · LB · UF · **LF**   | Lower Front |
| 3      | UB · LB · UF · LF · **UB**   | Upper Back |
| 4      | LB · UF · LF · UB · **LB**   | Lower Back |

O treino que abre e fecha a semana recebe uma sessão a mais — é a ênfase natural daquele ciclo de 4 semanas. Isso cria variação de volume cíclica sem que o praticante precise manipular nada: o macrociclo emerge da frequência escolhida.

### O mini deload estrutural

A rotatividade de ênfase entre semanas — mais Front em uma, mais Back na outra — funciona como um mecanismo de deload parcial e cíclico. Os grupos que receberam mais volume na semana anterior entram na semana seguinte com menos estímulo direto, favorecendo a recuperação sem interromper o treino.

Esse conceito tem suporte na literatura sobre periodização ondulatória semanal (WUP — Weekly Undulating Periodization) e nas evidências sobre deload planejado: períodos de volume reduzido inseridos no programa, mesmo que parciais, ajudam a controlar a fadiga acumulada e a sustentar a progressão a longo prazo.

## A fusão de PPL e Upper/Lower

O Wave é, tecnicamente, a fusão das duas divisões mais populares do treinamento de força: o PPL (Push/Pull/Legs) e o Upper/Lower.

**PPL** organiza o treino por padrão de movimento:
- Push (empurrar): peito, ombros, tríceps
- Pull (puxar): costas, bíceps
- Legs (pernas): quadríceps, posterior, glúteos, panturrilha

**Upper/Lower** organiza por região anatômica:
- Upper: todos os grupos do tronco superior
- Lower: todos os grupos do tronco inferior

O problema de usar qualquer um deles isoladamente é que cada um resolve bem um tipo de problema mas cria outros — o PPL tende a ter frequência baixa por grupo (1x/semana em muitas implementações), e o Upper/Lower pode criar sessões longas e sobrepostas de fadiga entre grupos.

O Wave não escolhe um nem outro. Ele usa a lógica de padrão de movimento do PPL para organizar o sequenciamento intrassessão, e usa a lógica de distribuição de frequência do Upper/Lower para garantir que cada grupo seja treinado 2x ou mais por semana. O resultado é uma estrutura de sessões que não se encaixa em nenhum dos dois formatos — porque é a intersecção matemática dos dois.

Essa é a resposta para a observação que motivou o projeto: a maioria dos treinadores chegava nas mesmas divisões não porque eram ótimas, mas porque eram os extremos mais conhecidos de um espectro. O Wave ocupa o meio desse espectro — de forma calculada.

## Por que "Wave"

Quando o modelo é visualizado em um gráfico com as sessões no eixo X e os grupos musculares no eixo Y, o padrão de volume e intensidade ao longo do tempo forma ondas. Grupos com maior prioridade oscilam com amplitude maior; grupos secundários têm ondas mais suaves. O gráfico gerado pelo próprio método deu nome ao projeto.

## O que o Wave não é

- Não é uma rotina pronta para seguir sem adaptação
- Não é um método de periodização linear clássica (não aumenta carga semana a semana de forma rígida)
- Não é uma lista aleatória de exercícios — as escolhas seguem critérios de ângulo, padrão de movimento e posição na sessão
- Não é para iniciantes (assume que o praticante já tem padrões motores estabelecidos)
